第八十八章 整理一下

CSS 知识的整理和总结。

到这里我们 CSS 部分就讲完了。事实上，CSS 的内容远比这要多，只是我们作为新手，初期先掌握这么多，已经够应付大部分问题了。其余的可以在熟练掌握，现在已有的知识之后，再慢慢的做补充。

还有一点内容，是我打算讲，但并不是现在讲的。

元素的变换，变形，旋转等，计划放在我们学过函数之后再讲。因为反正用到的也不多，而在学个函数之后再看这个属性，会变得更加容易理解，甚至是十分容易理解。

CSS3 的一个特点就是增加了函数处理。当然，不理解这点也没有关系，完全可以照常使用。如果能够理解什么是函数，这可以对整体的使用上有更进一步的掌握。

然后过渡效果和动画，我们放在学过 jQuery 之后再讲。一方面，而是有一些函数相关的东西，另一方面，他们需要一些触发的条件，或者流程控制。现在我们学也没有问题，但是可以应用的场景就很少。老也用不上，又比较复杂，肯定马上就忘掉了。

本书讲解的顺序都是以容易理解和掌握，为首要考虑因素，所以和文档类的教程区别会很大。文档类，则是方便查阅，遇到一个问题就把这个问题彻底讲完整。所以不应该放在一起比较。

目前为止，除了需要动手做练习，剩下应该没有什么痛苦的地方。如果你真的去动手做了练习，又去查阅了 Emmet 的用法。你会发现，唯一的痛苦就是需要动脑。而如果你做了几个页面的练习之后，你会发现它们是有规律的，熟悉之后基本都是无脑操作了，反倒是许多细节的调整比较费时间。总之，随着你的反复使用，你对这些东西的理解也会发生变化。

如果 CSS 部分还没有真的掌握，那么答应我，一定不要往下去看。因为后面的东西，是以前面这些知识为基础的。写十个页面没有你想象中的那么难，也许只是痛苦几天而已，在学习的道路上，偶尔需要咬牙几天，也不算是很过分的要求吧。

而后面的内容，你要答应我，一定要动手操作，而且要一边读一边操作，而且就算是已经会了，也再做几遍。我会努力让事情变得简单，而你也要努力跟上我的思路。

好了，话唠的小老鼠碎嘴完了，我们开启下一个征程吧~
