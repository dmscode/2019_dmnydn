第五章 如何标记

HTML 是一种标记语言，如何让标记和内容之间能够的到明确的区分，这需要一些规则来规范。现在我们来学习一下它的基本格式。

上一章我们讲了 HTML 究竟是个什么？所以很容易发现，在超文本标记语言这个名词里，我们真正要学习的就是标记。当然，我们已经学习了其中一个标记，就是“斯壮”（ <strong></strong>）。现在我们来做一个简单的分析：

首先，标记都是用尖括号（<>）括起来的，这是为了和内容进行区分，那么尖括号里面的就都是用来对内容进行解释说明的，而不是用来直接显示的。

然后这两对尖括号，一个里面有斜线（/），一个里面没有，没有斜线的是起始标签（<strong>），表示这个标记的开始；相对应的，有斜线的就是结束标签（</strong>），表示这个标记的结束。

而尖扩号里面会有一些文字，比如我们现在的标签，里面写着：strong，这个是标签的名字，我们叫做“标签名”，是用来说明这个标签是做什么的。

最后这两个标签之间的内容，就是被标记内容。一对标签，以及其中被标记的内容放在一起就是一个元素。

那么关于标签的基础知识，目前就是这些，细节部分我们等到后面再做补充，免得一次都说出来，大家就乱了。如果你觉得上面这样用文字描述有些混乱，那么下面我们用一张图来说明一下：

这样就很容易明白了，也无需刻意记忆什么，这只是为了我们以后交流起来方便，做一些约定。

那么，大家就发现了，这个核心的概念——标记，并不难以理解。而一旦理解这个问题之后，好像只要知道足够多的标记（标签），就可以去使用了。实际情况差不多也就是这样，所以后面我们主要就是介绍这些标签，以及几个特别的情况。
